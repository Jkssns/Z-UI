## Build Setup

``` bash
# install dependencies
npm install   // install个几把install

# serve with hot reload at localhost:8080
npm run dev  // 实际是npm run serve,如果不能运行建议使用 npm run dev

# build for production with minification
npm run build  // 有个屁用
```
